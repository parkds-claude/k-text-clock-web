# 인용구 기여 가이드

K-Text Clock은 한국 문학 속 시간 표현을 모읍니다.  
책을 읽다가 시각이 등장하는 문장을 발견하셨다면 기여해 주세요.

## 기여 조건

- **저작권 만료(PD) 작품만** — 저자 사후 70년 이상 경과
- 1956년 이전 사망한 한국 작가의 작품 (예: 이상, 김소월, 백석, 윤동주 등)
- 한자 고전, 조선왕조실록, 고전 시조·가사 환영

## 제출 방법

GitHub Issue를 열고 아래 형식으로 제출해 주세요:

```
시각: 19:00
문장: 저녁 일곱 시쯤 해서 하는 두 번째 세수는 손이 많이 간다.
출처: 이상, 「날개」, 1936
출처URL: https://ko.wikisource.org/wiki/날개
```

## 현재 비어있는 시간대

새벽 02:00~05:00 구간이 가장 부족합니다.  
밤샘하며 읽은 책 속 새벽 장면을 찾아주세요!
