# 영문 문학 시각 인용구 (English Literature Time Quotes)

시각이 등장하는 영문 문학 인용구 데이터입니다. 저작권 상태에 따라 두 파일로 분리되어
있으며, **라이선스가 서로 다릅니다.**

## quotes_en_pd.json — 77건 (한국어 번역 포함)

- 원작: Public Domain (저자 사후 70년 이상 — Dickens, Joyce, Conan Doyle 등)
- 원문 출처: Project Gutenberg (각 항목 `source_url`) 및 JohannesNE/literature-clock
- 한국어 번역: K-Text Clock 프로젝트의 AI 번역 (2차적 저작물)
- **라이선스: CC BY-NC 4.0** (이 레포 데이터 정책과 동일)

## quotes_en_litclock.json — 1,709건 (영어 원문만)

- 출처: [JohannesNE/literature-clock](https://github.com/JohannesNE/literature-clock) 데이터셋
- **라이선스: CC BY-NC-SA 2.5** (원 데이터셋 라이선스를 그대로 승계 — 비영리,
  동일조건변경허락)
- 인용된 원문 다수는 저작권이 살아 있는 현대 작품의 단문 발췌입니다. 원 프로젝트와
  동일하게 짧은 인용 범위로만 제공됩니다.

### 한국어 번역본이 없는 이유

이 1,709건의 원작은 생존·현대 작가의 저작권 보호 작품입니다. 번역은 원작의
2차적 저작물로 번역권이 원저작자에게 있어, **한국어 번역본은 공개하지 않습니다**
(단문 영어 원문 인용의 재배포와는 법적 성격이 다릅니다).
